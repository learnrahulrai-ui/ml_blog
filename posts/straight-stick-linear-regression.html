The Straight-Stick Rule: Setting the Dials, Checking the Leftovers, Getting the Numbers Right
Posted: 2026-06-01  |  Author: Rahul Rai  |  Tags: linear-regression, ols, least-squares, numerical-stability

The straight-stick rule guesses using a weighted sum of the measured columns plus a fixed nudge. Unlike the ask-closest rule it squeezes the whole working pile into d+1 dials, then discards the pile. The dials have an exact formula — no searching required.

The Rule

   one row of measured things:  [rooms, income, age, …]
         ×
   one dial per column:         [β₁,    β₂,    β₃,  …]
         +  fixed nudge β₀
         =  guess ŷ

   all rows at once:  ŷ = Xβ
   (X has a column of 1s prepended to carry β₀)
ŷi = β0 + Σj βj xij  ⇔  ŷ = Xβ

Each dial βj is the change in the guess per unit change in column j, all other columns held fixed. The columns can be squares, logs, or products of raw columns — the rule stays linear in the dials.

The Squared Leftover to Shrink

         ╲         ╱
          ╲       ╱        height  = total squared leftover J(β)
           ╲     ╱         bottom  = the best dials β
            ╲ • ╱

Pick the dials that shrink the squared leftover sum:

J(β) = ‖Xβ − y‖² = Σi (ŷi − yi)²

J is bowl-shaped — its Hessian 2XᵀX is never-negative — so any flat point is the global bottom. No local traps, no searching.

Finding the Dials: The Flat-Point Equations

Set the slope of the bowl to zero:

∇J = 2Xᵀ(Xβ − y) = 0  ⇒  XᵀXβ = Xᵀy  ⇒  β = (XᵀX)−1Xᵀy

Worked by hand at one column (d = 1) on three rows — Σ1, Σx, etc. are the entries of XᵀX and Xᵀy: 

   x = 1, 2, 3     y = 2, 3, 5
   Σ1 = 3   Σx = 6   Σy = 10   Σxy = 23   Σx² = 14

   XᵀX β = Xᵀy   →    3b +  6w = 10
                       6b + 14w = 23

   solve:  2w = 3  →  w = 3/2 ;   3b + 9 = 10  →  b = 1/3
   stick:  ŷ = 1/3 + (3/2)x

Checking the Leftovers

At the best dials, Xᵀ(y − Xβ) = 0, written Xᵀe = 0: the leftover column sits at right-angles to every measured column. Because one column is all ones, the leftovers also sum to zero. 

   x=1: ŷ=1.833, leftover = +0.167
   x=2: ŷ=3.333, leftover = −0.333
   x=3: ŷ=4.833, leftover = +0.167

   Σ leftover        = 0  ✓   (right-angles to the ones column)
   Σ x · leftover    = 0  ✓   (right-angles to x column)
   RMSE = √(mean leftover²) ≈ 0.236

Geometrically: ŷ = Hy where the flat-shadow thrower H = X(XᵀX)−1Xᵀ projects y onto the flat sheet spanned by the measured columns. H is symmetric and squaring it gives itself back (H² = H). trace(H) equals the number of fitted dials. No point on the flat sheet sits closer to y.

★ SAME WORK, BIGGER SHEET The three-row pencil solve is identical to what the toolbox runs on 16,000 rows and 8 columns — only the size of XᵀX changes. Fitted dials land in m.coef_, the fixed nudge in m.intercept_.

When the Straight-Stick Rule Is the Best Possible

The Gauss–Markov result: if the leftover has zero average given the rows (E[ε|X] = 0), the same spread everywhere, leftovers do not echo each other, and X has full column rank — then the straight-stick rule is BLUE: the lowest-spread rule among all flat, honest rules.

  • Bell-curve shape of leftovers is not required for BLUE. You need it only for exact confidence ranges and t-tests, not for the rule to be the best flat rule.
  • Full column rank means no measured column is a recipe of the others. If one is, XᵀX is flat (no single solution) and (XᵀX)−1 does not exist.

Getting the Dials Right: Don't Flip the Sheet

   build XᵀX: O(nd²)   ← n rows, each d² multiplications
   solve for β: O(d³)
   total: O(nd² + d³)

Do not compute (XᵀX)−1 by flipping the sheet directly. Forming XᵀX squares the shakiness number (κ(XᵀX) = κ(X)²) and loses precision. Instead:

  • Solve the flat-point equations directly (Cholesky cut of XᵀX), or better,
  • Cut X itself with QR or SVD and solve without ever forming XᵀX. The toolbox's LinearRegression uses an SVD-based approach.
  • For very many rows or a stream, roll downhill a little at a time: O(nd) per step instead of a full cut.
  • If XᵀX is flat or shaky (one column is a near-recipe of another), add a small bump to the diagonal — the ridge rule: β = (XᵀX + λI)−1Xᵀy — solvable for any λ > 0.
▸ SAME-RULER CONTRAST WITH ASK-CLOSEST RULE The straight-stick rule does not care which ruler you used. Rescale a column and its dial rescales inversely — guesses and RMSE stay the same. (The ask-closest rule breaks without same-ruler treatment.) You do need same-ruler when rolling downhill or when adding a bump, because those care about the size of the dials, which depends on scale.

In the Toolbox

from sklearn.linear_model import LinearRegression
from sklearn.metrics import mean_squared_error

m = LinearRegression()          # SVD-based flat-point solve under the hood
m.fit(X_train, y_train)         # sets dials: XᵀX β = Xᵀy
pred = m.predict(X_test)        # one dot-product per row

rmse = mean_squared_error(y_test, pred) ** 0.5
print(round(rmse, 3), m.coef_, m.intercept_)

Ask-Closest vs. Straight-Stick

 Ask-Closest Rule (KNN)Straight-Stick Rule (OLS)
What it storeswhole piled+1 dials only
Build costO(1) (store only)O(nd²+d³) (solve equations)
Guess costO(n·d) per rowO(d) per row
Setting to pick by handk (by rotating folds)none (λ if adding a bump)
Same-ruler needed?yes — requiredno — not for plain rule
Shape of guess surfacepiecewise-flat, localone flat sheet, global

Opposite approaches, one question: given the working rows, guess the right answer for a new row. Ask-closest leans on stored rows; straight-stick compresses them into a flat sheet.

Going Deeper

  • flat-point equations, ordinary least squares — the solve
  • QR / SVD least squares, shakiness number (condition number) — the numerics
  • flat-shadow thrower (hat matrix), right-angle leftovers — the geometry
  • Gauss–Markov, unequal leftover spread (heteroscedasticity) — the assumptions
  • ridge, lasso — when columns are near-recipes of each other
  1. Rule: ŷ = Xβ; set dials by shrinking ‖Xβ−y‖².
  2. Bowl-shaped mistake → exact dials from β = (XᵀX)−1Xᵀy.
  3. Leftovers sit at right-angles to measured columns; ŷ = Hy is a flat-sheet shadow.
  4. Straight-stick rule is BLUE under Gauss–Markov; bell-curve leftovers needed only for tests.
  5. Cut the sheet (QR/SVD), don't flip it; add a bump when XᵀX is flat.
